# 兼容性与常见问题

## 支持设备

LeviLauncher 面向 Android 9.0 或更高版本的 ARM64 设备。可用内存和存储空间越充足，管理多版本、大型世界和资源包时越稳定。

## Minecraft 要求

LeviLauncher 要求设备上安装来自 Google Play 的官方 Minecraft 基岩版。它面向合法玩家，不提供 Minecraft 授权。

如果 Minecraft 未安装、来源不受支持，或版本过旧，启动器可能会在启动游戏前停止。

## Minecraft 版本

当前应用不支持早于 1.21.80 的 Minecraft 版本。导入版本如果需要独立数据，应启用版本隔离。

## 存储与迁移

启动器可能需要先迁移旧数据目录才能继续使用。迁移成功并确认世界、资源包、设置可用之前，不要删除旧目录。

备份会保存在启动器管理的备份位置。重要世界建议额外保留一份在应用数据目录之外。

## Native 模块

Native 模块可以扩展游戏，但也会在游戏进程中运行本机代码。只安装你信任来源的模块，并优先在隔离版本中测试。

## 常见问题

### 为什么提示 Minecraft 未安装？

请从 Google Play 安装或更新官方 Minecraft，然后重新打开 LeviLauncher。

### 为什么导入版本启动失败？

导入版本通常需要版本隔离。进入实例设置启用版本隔离后再试。

### 开发者 API 问题应该看哪里？

请阅读 [开发者](/zh-CN/guide/developer) 分类。Preloader API 参考刻意放在该分类下，而不是普通用户指南里。
